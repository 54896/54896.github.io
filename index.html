Lego robotti - breikkaaja

ohjelma liikuttaa robotin jalkoja ja käsiä samaan aikaa, hubissa pyörii numerot 1-3, joka liikkumiskierroksen jälkeen numero kasvaa, kun tulee numero 3, se aloittaa uudelleen numerosta 1 from hub import light_matrix, port import runloop, motor async def main(): await motor.run_to_absolute_position(port.F, 0, 720) runloop.sleep_ms(1000) count = 1 # aloittaa numerosta 1 for i in range(10): # tekee toiminnon 10 kertaa / kierrosten määrää pystyy muuttamaan if count == 4: # jos numero menee neljä / numeroa pystyy muuttamaan = laskee haluttuun numeroon asti (esim. 100) count = 1 # se pyörähtää takaisin 1 light_matrix.write(str(count)) # numerot näkyy hubissa count += 1 # sama, kuin count = count + 1 motor.run_for_degrees(port.F, -360, 720) # liikuttaa jalkoja await motor.run_for_degrees(port.D, -360, 720) # liikuttaa käsiä runloop.sleep_ms(1000) runloop.run(main()) # ajaa koodin hubiin
